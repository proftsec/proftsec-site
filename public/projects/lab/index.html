Fundamentals of Linux#

Purpose#

The Linux operating system is widely used in cybersecurity and IT infrastructure. It offers numerous security auditing tools not available on Windows. This project introduces students to Linux, allowing them to become comfortable with basic navigation, file operations, and essential commands for security analysis.

Virtualization Options#

To complete this assignment, you will need access to a Linux environment. You can choose one of the following options:

  • VirtualBox (Recommended): Install and run a Debian VM on your local machine.
  • VMware Workstation Player: Similar to VirtualBox but may require a license for advanced features.
  • WSL 2 (Windows Subsystem for Linux): If using Windows, you can install Debian through WSL without needing a full VM.
  • AWS Free Tier: Use a cloud-based Debian instance if you prefer remote access.
  • Microsoft Azure (Student Credits): Free VM access for students.

Expectations#

After this project, you will be able to:

  • Navigate the Linux file system.
  • Copy, delete, and move files/directories.
  • Search for text within files.
  • Retrieve network configuration details.
  • Edit files using Nano.
  • Identify logged-in users.

Directions#

Setting Up Your Linux Environment#

Follow these steps to set up your Debian system:

  1. Download and Install VirtualBox

    • Install VirtualBox and create a new VM.
    • Download the Debian ISO and install it on your VM.

  2. Boot into Debian Linux

    • Power on the VM and log in.
    • Open a terminal window.

Basic Linux Commands#

1. Checking Your IP Address#

ip address show

This command displays network details, including IP addresses and interface names.

2. Testing Network Connectivity#

ping -c 4 www.google.com

This sends four ICMP packets to Google‚Äôs server to test network connectivity.

3. Viewing Logged-In Users#

who

Lists currently logged-in users.

4. Navigating the File System#

pwd  # Shows current directory
ls -l  # Lists files with details
cd /home  # Changes to the home directory

5. Creating and Managing Files#

touch myfile.txt  # Creates an empty file
echo "Hello, Linux!" > myfile.txt  # Writes text to a file
cat myfile.txt  # Displays file content
mv myfile.txt newfile.txt  # Renames the file
cp newfile.txt copyfile.txt  # Copies the file
rm copyfile.txt  # Deletes the file

6. Editing Files with Nano#

nano myfile.txt
  • Modify text and press Ctrl + X, then Y to save and exit.

7. Using Shell Scripting#

nano count.sh

Add the following script:

#!/bin/bash
for i in 1 2 3
do
echo $i
done

Save the file and grant execute permissions:

chmod +x count.sh
./count.sh  # Runs the script

Organizing Files into Directories#

mkdir mydir
mv newfile.txt mydir/
ls mydir/

Additional Notes#

  • To shut down your VM: sudo shutdown -h now
  • To restart: sudo reboot
  • Check command history: history
  • View system logs: dmesg | tail
